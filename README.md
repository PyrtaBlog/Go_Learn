# 🚀 RestStream API

Минималистичный высокопроизводительный REST API на **Go**, предназначенный для приема потоковых данных и их передачи в **Apache Kafka**.

## 🎯 Суть проекта
Сервис служит «входными воротами» (Ingress) для аналитических данных. Он принимает HTTP-запросы, обогащает их метаданными, проверяет структуру и подготавливает для асинхронной обработки.

## 🛠 Технологический стек
- **Runtime**: Go 1.21+ (используем `slices` и нативный `net/http`)
- **Logging**: [Logrus](https://github.com) для структурированных JSON-логов
- **Format**: [gofmt](https://pkg.go.dev) — стандарт форматирования

## ⚙️ Команды управления (Makefile)

Проект использует автоматизацию через `make`. Цепочка команд гарантирует, что код всегда отформатирован и проверен перед запуском.


| Команда | Описание | Порядок действий |
| :--- | :--- | :--- |
| `make fmt` | Форматирует код всего проекта | — |
| `make vet` | Запускает статический анализатор | `fmt` → `vet` |
| `make run` | **(Default)** Запускает приложение | `fmt` → `vet` → `run` |

**Пример запуска:**
```bash
make
# или
make run
